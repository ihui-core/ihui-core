IHUI CORE

Fuente de verdad institucional.

Objetivo:

Conectar herramientas existentes sin reemplazarlas.

No es ERP.
No es CRM.
No es Teams.
No es Slack.
No es Notion.

IHUI CORE almacena hechos.

Ejemplos:

- Caso creado
- Documento recibido
- Tarea asignada
- Decisión tomada
- Incidente reportado

MVP:

- Personas
- Casos
- Tareas
- Documentos
- Actividad

Pantalla principal:

- Buscar
- Mis tareas
- Casos abiertos
- Actividad reciente

Arquitectura:

- Next.js
- PostgreSQL
- Tailwind

Misma experiencia visual que IHUI Notarías y RHM.

Futuro:

Integración con Teams, Outlook, WhatsApp, GitHub, Asana y otras herramientas.

La fuente de verdad guarda hechos, no conversaciones.
